# Stock Trading Platform

## Description
This project is a Java-based Stock Trading Platform developed as part of a Java Programming Internship. It simulates basic stock market operations and allows users to buy and sell stocks through a menu-driven console application.

## Features
- View available stocks and prices
- Buy stocks
- Sell stocks
- Manage portfolio
- Track account balance
- User-friendly menu-driven interface

## Technologies Used
- Java
- OOP Concepts
- HashMap
- Scanner Class

## How to Run
1. Open the project in any Java IDE.
2. Compile the program.
3. Run the StockTradingPlatform.java file.
4. Follow the menu options displayed on the console.
